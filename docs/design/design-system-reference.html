Image Manager

Foundations and the upload dropzone treatment — clarity, deference, depth. Near-monochrome UI, one accent, content stays the hero.

Notes for review
  • Single accent (blue) reserved for selection, links, and primary actions only — everything else stays neutral gray so photos read as the hero.
  • System font stack (SF Pro on Apple devices, falls back gracefully elsewhere) at HIG type-scale steps.
  • Depth comes from soft, low-opacity shadows and translucency, not borders — thin hairline borders only where a hard edge is needed (sidebar, input fields).
  • Toggle light/dark above — every token below is theme-aware.
